Błąd z pustym plikiem Json

Aby ten błąd naprawić musisz wejść do folderu bota a następnie wejść w folder Data i wejść w plik który wskazuje błąd. Jeśli jest pusty dodaj {}



Błąd z Intentami

Jeśli wyświetla ci się w logach błąd o nie włączonych intentach wejdź na stronę Discord developer portal następnie wybierz swojego bota wejdź w Bot (na pasku po lewej stronie) zjedź niżej i zaznacz wszystkie elementy które znajdują sie w nagłówku Privileged Gateway Intents

Błąd z lokalizacja

Jeśli chcesz odpalić bota i wyskakuje ci błąd Error: Cannot find module (ścieżka) to oznacza to, że próbujesz wpisać node . w złej lokalizacji. Aby to naprawić musisz zrobić jak na tym poradniku

Błąd z brakującym modułem

Jeśli wyskakuje ci błąd typu Cannot find module 'modul' wejdź w folder bota i na tym największym pasku wpisz cmd i wpisz tam npm i modul jesli nie masz node zrob to jak na tym poradniku

Aktualizacja djs do nowszej wersji

Aby to zrobić musimy wejść do folderu bota a następnie na tym dużym pasku wpisujemy cmd. Aktualnie zalecane jest mieć wersje 13.7.0. Aby zaktualizować wpisz npm i discord.js@13.7.0 bądź npm i discord.js@wersja
