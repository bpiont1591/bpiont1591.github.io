Jeśli wyskakuje ci błąd typu

"node.' is not recognized as an internal or external command, operable program or batch file."

Wejdź na stronę Node.js (klik) i pobierz wersję 18.15.0 przy instalacji zaznacz npm package

Jeśli zainstalowaliśmy node.js upewnij się czy aby napewno masz zainstalowane. Wejdź w cmd komputera i wpisz node -v i npm -v

Jeśli wyświetla nam się wersja node i npm to oznacza, że node.js zostało zainstalowane.

*UWAGA jeśli instalujemy node.js czekamy do samego końca aż odpali się PowerShell
